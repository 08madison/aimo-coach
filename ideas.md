# 奥数AI教练 — 设计方案

## 三个候选方向

### 方向A：极光学院 (Aurora Academy)
清新的蓝绿渐变配色，带有几何数学符号的动态背景，像极光一样流动。现代感强，激励学生探索。
概率：0.07

### 方向B：纸质笔记本 (Notebook Warmth)
米白色背景，手写字体点缀，像一本精心整理的数学笔记本。温暖、亲切、低压力。
概率：0.05

### 方向C：清新数字空间 (Fresh Digital Space) ← 选定
清爽的白色+翠绿+深蓝配色，干净的卡片式布局，带有柔和的数学符号装饰。既专业又对学生友好，不会有考试压力感。
概率：0.08

---

## 选定方案：清新数字空间 (Fresh Digital Space)

### Design Movement
Clean Tech meets Educational Warmth — 参考 Linear、Notion 的极简主义，融入教育产品的温暖感

### Core Principles
1. **呼吸感布局**：大量留白，内容不拥挤，让学生专注于题目本身
2. **数学符号作为装饰**：将 ∑、∫、π、∞ 等符号作为轻量背景装饰，营造数学氛围而不压迫
3. **进度可视化**：用绿色渐变进度条和分数徽章强化成就感
4. **对话式交互**：AI教练的回复像朋友聊天，不像冷冰冰的系统

### Color Philosophy
- **主色**：翠绿 `#10B981` (Emerald-500) — 代表成长、正确、活力
- **辅色**：深蓝 `#1E40AF` (Blue-800) — 代表知识、专业、深度
- **背景**：纯白 `#FFFFFF` + 极浅灰 `#F8FAFC`
- **强调**：琥珀黄 `#F59E0B` — 用于高亮提示和星级评分
- 情感意图：让学生感到"我能做到"，而不是"这太难了"

### Layout Paradigm
- 顶部固定导航 + 左侧学科分类 + 右侧主内容区
- 首页：不对称英雄区（左文字右插图），下方特性网格
- 练习页：左侧题目 + 右侧AI对话面板（分屏布局）
- 订阅页：三栏卡片对比

### Signature Elements
1. **数学符号浮动背景**：半透明的 π、∑、∞ 符号轻浮在背景中
2. **绿色进度环**：每道题完成后的圆形进度动画
3. **AI教练头像**：一个戴眼镜的友善机器人图标

### Typography System
- **标题**：Noto Serif SC（衬线，权威感）
- **正文**：Noto Sans SC（无衬线，清晰易读）
- **数学公式**：KaTeX 渲染
- 层级：H1 2.5rem bold / H2 1.75rem semibold / Body 1rem regular

### Brand Essence
"让每个孩子都有专属的奥数名师" — 面向7-12年级竞赛生，AI驱动的个性化辅导
个性形容词：专业、温暖、激励

### Brand Voice
- 标题示例："你的专属奥数教练，24小时在线"
- CTA示例："开始今日挑战 →"
- 禁止使用："欢迎来到我们的网站"、"立即注册"等泛化文案

### Signature Brand Color
翠绿 `#10B981` — 成长绿，专属于奥数AI教练

## Style Decisions
- 使用 Noto Serif SC + Noto Sans SC 字体组合
- 主色调：Emerald绿 + 深蓝，背景白色
- 卡片使用轻微阴影（shadow-sm）而非边框
- 按钮 active 状态 scale(0.97) 动效
- 数学符号作为装饰性背景元素
